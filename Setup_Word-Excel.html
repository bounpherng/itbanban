ຄູ່ມືການຕິດຕັ້ງ ແລະ ໃຊ້ງານ Add-in

ວິທີແກ້ໄຂບັນຫາ Setup ແລະ ການນຳໃຊ້ Word/Excel Add-in

ແກ້ໄຂບັນຫາຕິດຕັ້ງບໍ່ໄດ້

ບັນຫາການຕິດຕັ້ງ ແລະ ວິທີແກ້ໄຂ

ຫາກທ່ານດາວໂຫຼດໄຟລ໌ Setup_Word.exe ຫຼື Setup_Excel.exe ມາແລ້ວ ແຕ່ບໍ່ສາມາດຕິດຕັ້ງໄດ້ ຫຼື ຖືກລະບົບປ້ອງກັນ (Antivirus) ລົບຖິ້ມ:

1. ປິດ Real-time Protection ຊົ່ວຄາວ

Windows Defender ອາດຈະເຂົ້າໃຈຜິດວ່າໄຟລ໌ເປັນໄວຣັດເນື່ອງຈາກເປັນໂປຣແກຣມທີ່ຂຽນຂຶ້ນເອງ (Unknown Publisher).

  • ໄປທີ່ Settings > Update & Security
  • ເລືອກ Windows Security > Virus & threat protection
  • ກົດ Manage settings ແລະປິດ Real-time protection
2. ວິທີກົດ Run Anyway

ເມື່ອເປີດໄຟລ໌ແລ້ວຂຶ້ນໜ້າຈໍສີຟ້າ (Windows protected your PC):

1. ຄິກທີ່ຂໍ້ຄວາມຂີດກ້ອງ More info
2. ປຸ່ມ Run anyway ຈະປະກົດຂຶ້ນມາ ໃຫ້ກົດປຸ່ມນັ້ນເພື່ອຕິດຕັ້ງ.
